# 🤖 Digital Detox Bot — 30-дневный марафон по избавлению от зависимости от соцсетей

**Digital Detox Bot** — это Telegram-бот, который помогает пользователю постепенно сокращать время в социальных сетях и формировать здоровые цифровые привычки через 30-дневный марафон.

Бот мотивирует, напоминает и отслеживает прогресс, превращая отказ от бессмысленного скроллинга в понятный и управляемый процесс.

---

## 📌 Функциональность

* 🗓 **30-дневный марафон**
  Каждый день новые задания для снижения времени, проведённого в соцсетях.

* 🔔 **Напоминания и мотивация**
  Бот отправляет уведомления и мотивирующие сообщения, чтобы поддерживать пользователя.

* 📊 **Отслеживание прогресса**
  Возможность видеть свой прогресс и достижения за весь марафон.
  
---

## 🛠 Технологии

* Python 3.x
* [aiogram](https://docs.aiogram.dev/) — библиотека для создания Telegram-ботов
* SQLite  — база данных для хранения прогресса


---

## 📈 Как использовать

1. Откройте бота в Telegram и отправьте команду `/start`.
2. Получайте ежедневные задания и мотивационные уведомления.
3. Проверяйте свой прогресс с помощью команды `/progress`.
4. Следуйте заданиям и постепенно сокращайте время в соцсетях.
